# Primality Tests:
* Miller - Rabin
* Sieve of Atkin
* Deterministic

Compile:
```
g++ -std=c++11 -o prime_algorithms prime_algorithms.cpp
```
The code might run faster if you compile with optimization parameters  
https://gcc.gnu.org/onlinedocs/gcc/Optimize-Options.html

Run:
```
./prime_algorithms
```
